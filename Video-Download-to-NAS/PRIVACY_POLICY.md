# Privacy Policy

**Last Updated: November 24, 2025**

## 1. Introduction

This Privacy Policy describes how Video Download to NAS (VDTN) handles information when you use our software.

## 2. Information We Do NOT Collect

VDTN is designed with privacy in mind. We do NOT collect:
- Personal information (name, email, address)
- Usage statistics or analytics
- Downloaded content information
- User behavior data
- Device information
- IP addresses or location data
- Any data transmitted to external servers

## 3. Local Operation

### 3.1 Self-Hosted Software

VDTN operates entirely on your local infrastructure:
- All processing happens on your device/server
- No data is transmitted to our servers
- No external connections except for downloading content
- No tracking or monitoring capabilities

### 3.2 Data Storage

All data remains under your control:
- Downloaded content stays on your designated storage
- Configuration files stored locally
- No cloud storage or external databases
- You have complete control over all data

## 4. Third-Party Services

### 4.1 yt-dlp Integration

VDTN uses yt-dlp to download content:
- yt-dlp may connect to video platforms directly
- We do not control yt-dlp's data handling
- Refer to yt-dlp's documentation for their practices
- No data is shared between VDTN and yt-dlp developers

### 4.2 Video Platforms

When downloading content:
- Your IP address is visible to the source platform
- Platform-specific terms and privacy policies apply
- We do not control platform data collection
- Consider using VPN for additional privacy

## 5. Optional Features

### 5.1 SSO Authentication (Optional)

If you enable SSO features:
- OAuth tokens are stored locally only
- No user data is transmitted to VDTN developers
- Third-party OAuth providers have their own privacy policies
- You can disable SSO at any time

### 5.2 Logging (Local Only)

Application logs may contain:
- Download URLs and status
- Error messages and debugging information
- System performance data
- All logs are stored locally only

## 6. Data Security

### 6.1 Local Security

Your data security depends on:
- Your local system security
- Network configuration
- Access controls you implement
- Regular security updates

### 6.2 Recommendations

We recommend:
- Using strong passwords for web interface
- Enabling HTTPS in production
- Restricting network access appropriately
- Regular system updates

## 7. Data Retention

### 7.1 User Control

You control all data retention:
- Delete downloaded content anytime
- Clear logs as needed
- Uninstall software completely removes all traces
- No external data to delete

## 8. Children's Privacy

### 8.1 No Age Restrictions

This software has no built-in age restrictions, but:
- Parents/guardians are responsible for children's use
- Ensure children understand legal responsibilities
- Monitor children's downloading activities
- Consider parental controls on your network

## 9. International Users

### 9.1 Global Use

This software can be used worldwide:
- No data crosses borders through our systems
- Local laws apply to your use
- Consider local privacy regulations
- Data remains in your jurisdiction

## 10. Changes to Privacy Policy

### 10.1 Updates

We may update this policy:
- Changes will be reflected in software documentation
- Continued use implies acceptance
- Major changes will be prominently noted
- Check documentation periodically

## 11. Your Rights

### 11.1 Complete Control

You have the right to:
- Access all your data (it's on your system)
- Modify or delete any data
- Export data in any format
- Stop using the software anytime

### 11.2 No External Data

Since we don't collect data:
- No data requests needed
- No account deletion process
- No data portability issues
- Complete privacy by design

## 12. Contact Information

### 12.1 Questions

For privacy-related questions:
- Refer to project documentation
- Check community forums
- Review source code (open source)
- No personal data to inquire about

## 13. Compliance

### 13.1 Regulations

This software is designed to comply with:
- GDPR (no data collection)
- CCPA (no data sale/collection)
- Other privacy regulations (by not collecting data)

### 13.2 Verification

You can verify our privacy claims by:
- Reviewing the open-source code
- Monitoring network traffic
- Checking local file system
- Using network analysis tools

---

**Summary**: VDTN respects your privacy by not collecting any data. Everything stays on your system, under your control. Your privacy is protected by design, not just by policy.
