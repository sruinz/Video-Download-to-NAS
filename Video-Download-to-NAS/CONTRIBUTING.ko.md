**이것은 개인 사용을 위해 유지 관리되는 기능 완성 개인 프로젝트입니다.**

### 의미

- ✅ **오픈 소스**: 누구나 사용, 연구 및 포크할 수 있는 코드

### 권장 접근 방식

이 프로젝트를 사용하거나 수정하려면:

1. **저장소 포크** - 자신의 사본 생성
2. **변경 수행** - 필요에 맞게 사용자 정의
3. **포크 유지 관리** - 적절하다고 생각되는 대로 업데이트 유지
4. **포크 공유** - 다른 사람이 개선 사항의 혜택을 받을 수 있음

이 접근 방식은 완전한 제어권을 제공하며 이 저장소가 적극적으로 유지 관리되는 것에 의존하지 않습니다.

## 법적 준수

### 중요 고지

이 프로젝트에 기여함으로써, 귀하는 다음을 인정합니다:

1. **도구 목적**: 이것은 범용 미디어 다운로드 도구입니다
2. **사용자 책임**: 사용자는 자신의 행동에 책임이 있습니다
3. **책임 없음**: 기여자는 사용자가 소프트웨어를 사용하는 방법에 대해 책임지지 않습니다
4. **합법적 사용**: 도구는 합법적이고 정당한 목적으로만 사용됩니다

### 기여 가이드라인

기여 시 다음을 확인하세요:

- ✅ 귀하의 코드가 불법 활동을 장려하지 않음
- ✅ 문서가 합법적이고 책임 있는 사용을 강조함
- ✅ 기능이 합법적 사용 사례를 지원함
- ✅ 하드코딩된 콘텐츠 소스 또는 저작권 자료 없음

## 개발 설정

### 사전 요구사항

- Docker & Docker Compose
- Python 3.11+
- Node.js 18+
- Git

### 로컬 개발

1. **저장소 복제**
   ```bash
   git clone https://github.com/sruinz/Video-Download-to-NAS.git
   cd Video-Download-to-NAS
   ```

2. **백엔드 설정**
   ```bash
   cd backend
   pip install -r requirements.txt
   uvicorn app.main:app --reload
   ```

3. **프론트엔드 설정**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
