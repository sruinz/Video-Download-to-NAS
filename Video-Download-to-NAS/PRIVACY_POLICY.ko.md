# 개인정보 처리방침

**최종 업데이트: 2025년 11월 24일**

## 1. 소개

이 개인정보 처리방침은 Video Download to NAS (VDTN) 소프트웨어 사용 시 정보 처리 방법을 설명합니다.

## 2. 수집하지 않는 정보

VDTN은 개인정보 보호를 염두에 두고 설계되었습니다. 다음을 수집하지 **않습니다**:
- 개인 정보 (이름, 이메일, 주소)
- 사용 통계 또는 분석
- 다운로드한 콘텐츠 정보
- 사용자 행동 데이터
- 기기 정보
- IP 주소 또는 위치 데이터
- 외부 서버로 전송되는 모든 데이터

## 3. 로컬 운영

### 3.1 셀프 호스팅 소프트웨어

VDTN은 전적으로 귀하의 로컬 인프라에서 작동합니다:
- 모든 처리는 귀하의 기기/서버에서 발생
- 우리 서버로 데이터가 전송되지 않음
- 콘텐츠 다운로드를 제외한 외부 연결 없음
- 추적 또는 모니터링 기능 없음

### 3.2 데이터 저장

모든 데이터는 귀하의 통제 하에 있습니다:
- 다운로드한 콘텐츠는 지정된 저장소에 유지
- 구성 파일은 로컬에 저장
- 클라우드 저장소 또는 외부 데이터베이스 없음
- 모든 데이터에 대한 완전한 통제권 보유

## 4. 제3자 서비스

### 4.1 yt-dlp 통합

VDTN은 콘텐츠 다운로드를 위해 yt-dlp를 사용합니다:
- yt-dlp는 비디오 플랫폼에 직접 연결할 수 있음
- yt-dlp의 데이터 처리를 통제하지 않음
- yt-dlp의 관행에 대해서는 해당 문서 참조
- VDTN과 yt-dlp 개발자 간에 데이터가 공유되지 않음

### 4.2 비디오 플랫폼

콘텐츠 다운로드 시:
- 귀하의 IP 주소가 소스 플랫폼에 표시됨
- 플랫폼별 이용약관 및 개인정보 처리방침 적용
- 플랫폼 데이터 수집을 통제하지 않음
- 추가 개인정보 보호를 위해 VPN 사용 고려

## 5. 선택적 기능

### 5.1 SSO 인증 (선택사항)

SSO 기능을 활성화하는 경우:
- OAuth 토큰은 로컬에만 저장됨
- VDTN 개발자에게 사용자 데이터가 전송되지 않음
- 제3자 OAuth 제공자는 자체 개인정보 처리방침을 가짐
- 언제든지 SSO를 비활성화할 수 있음

### 5.2 로깅 (로컬만)

애플리케이션 로그에는 다음이 포함될 수 있습니다:
- 다운로드 URL 및 상태
- 오류 메시지 및 디버깅 정보
- 시스템 성능 데이터
- 모든 로그는 로컬에만 저장됨

## 6. 데이터 보안

### 6.1 로컬 보안

귀하의 데이터 보안은 다음에 달려 있습니다:
- 귀하의 로컬 시스템 보안
- 네트워크 구성
- 구현한 접근 제어
- 정기적인 보안 업데이트

### 6.2 권장사항

다음을 권장합니다:
- 웹 인터페이스에 강력한 비밀번호 사용
- 프로덕션 환경에서 HTTPS 활성화
- 네트워크 접근을 적절히 제한
- 정기적인 시스템 업데이트

## 7. 데이터 보존

### 7.1 사용자 통제

모든 데이터 보존을 통제합니다:
- 언제든지 다운로드한 콘텐츠 삭제
- 필요에 따라 로그 지우기
- 소프트웨어 제거 시 모든 흔적 완전히 제거
- 삭제할 외부 데이터 없음

## 8. 어린이 개인정보 보호

### 8.1 연령 제한 없음

이 소프트웨어에는 내장된 연령 제한이 없지만:
- 부모/보호자가 어린이의 사용에 책임이 있음
- 어린이가 법적 책임을 이해하도록 보장
- 어린이의 다운로드 활동 모니터링
- 네트워크에서 자녀 보호 기능 고려

## 9. 국제 사용자

### 9.1 전 세계 사용

이 소프트웨어는 전 세계적으로 사용할 수 있습니다:
- 우리 시스템을 통해 국경을 넘는 데이터 없음
- 귀하의 사용에 현지 법률 적용
- 현지 개인정보 보호 규정 고려
- 데이터는 귀하의 관할권에 유지됨

## 10. 개인정보 처리방침 변경

### 10.1 업데이트

이 정책을 업데이트할 수 있습니다:
- 변경사항은 소프트웨어 문서에 반영됨
- 계속 사용하면 수락을 의미함
- 주요 변경사항은 눈에 띄게 표시됨
- 정기적으로 문서 확인

## 11. 귀하의 권리

### 11.1 완전한 통제

귀하는 다음 권리가 있습니다:
- 모든 데이터 접근 (귀하의 시스템에 있음)
- 모든 데이터 수정 또는 삭제
- 모든 형식으로 데이터 내보내기
- 언제든지 소프트웨어 사용 중지

### 11.2 외부 데이터 없음

데이터를 수집하지 않으므로:
- 데이터 요청 필요 없음
- 계정 삭제 프로세스 없음
- 데이터 이동성 문제 없음
- 설계상 완전한 개인정보 보호

## 12. 연락처 정보

### 12.1 질문

개인정보 보호 관련 질문:
- 프로젝트 문서 참조
- 커뮤니티 포럼 확인
- 소스 코드 검토 (오픈 소스)
- 문의할 개인 데이터 없음

## 13. 규정 준수

### 13.1 규정

이 소프트웨어는 다음을 준수하도록 설계되었습니다:
- GDPR (데이터 수집 없음)
- CCPA (데이터 판매/수집 없음)
- 기타 개인정보 보호 규정 (데이터 수집하지 않음으로써)

### 13.2 검증

다음을 통해 개인정보 보호 주장을 검증할 수 있습니다:
- 오픈 소스 코드 검토
- 네트워크 트래픽 모니터링
- 로컬 파일 시스템 확인
- 네트워크 분석 도구 사용

---

**요약**: VDTN은 데이터를 수집하지 않음으로써 귀하의 개인정보를 존중합니다. 모든 것이 귀하의 시스템에 유지되며 귀하의 통제 하에 있습니다. 귀하의 개인정보는 정책이 아닌 설계로 보호됩니다.
